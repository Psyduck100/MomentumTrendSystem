# PMTL Gold SMA Strategy

This strategy trades Gold (GLD) using a simple moving average (SMA) filter:

- **Asset:** GLD (SPDR Gold Shares)
- **Rule:** If GLD's price is above its x-day SMA at the start of the month, hold GLD; otherwise, hold cash.
- **SMA Window:** Parameterized (default: 100 days, configurable)
- **Rebalance:** First trading day of each month
- **No lookforward bias:** All signals use only data available up to the rebalance date.

## Usage

1. Ensure you are in the project root and using pipenv:

   ```sh
   pipenv run python pmtl_gold_sma/rebalance.py
   ```

2. To change the SMA window, edit the `SMA_WINDOW` variable in `rebalance.py` or call `run_strategy(sma_window=XXX)` in code.

## Requirements

- Python 3.8+
- pandas
- yfinance

Install dependencies (if needed):

```sh
pipenv install pandas yfinance
```

## Output

Prints a table of monthly signals, price, SMA, and action (hold GLD or cash).

---

**No lookforward bias:**

- The SMA is calculated using only data up to and including the rebalance date.
- The rebalance occurs on the first trading day of each month.
